# Ad Copy Frameworks

Universal frameworks for structuring ad scripts and copy. Each framework includes timing for video ads (30-60 seconds) and copy structure for static/image ads.

---

## PAS -- Problem, Agitate, Solution

Best for: cold audiences, pain-aware prospects, lead gen

### Video Script Template (30s)
```
[0-3s]  HOOK: State the problem they are living with right now
[3-10s] PROBLEM: Describe the situation -- make them feel seen
[10-18s] AGITATE: Show what happens if they do nothing -- the cost of inaction
[18-25s] SOLUTION: Introduce what you offer as the answer
[25-30s] CTA: Tell them exactly what to do next
```

### Copy Template
```
Line 1: [Problem statement as a question or bold claim]
Line 2: [Agitate -- what it costs them]
Line 3: [Solution -- what changes]
Line 4: [CTA -- one clear action]
```

### Example Structure
```
Hook: "Still spending 3 hours a day on [task] that should take 10 minutes?"
Problem: "Most [audience] are stuck doing [painful thing] manually..."
Agitate: "That is 60 hours a month you will never get back. Meanwhile your competitors..."
Solution: "We built a system that does [result] in [timeframe]."
CTA: "Book a free call and we will show you how it works."
```

## AIDA -- Attention, Interest, Desire, Action

Best for: warm audiences, product demos, higher-ticket offers

### Video Script Template (45s)
```
[0-3s]  ATTENTION: Pattern interrupt -- something unexpected or bold
[3-12s] INTEREST: Share a surprising fact, result, or insight
[12-25s] DESIRE: Paint the picture of life after they buy -- specific outcomes
[25-35s] ACTION: Clear CTA with urgency or scarcity if genuine
```

### Copy Template
```
Line 1: [Attention-grabbing opener -- stat, question, bold claim]
Line 2: [Interest -- the insight or mechanism]
Line 3: [Desire -- specific outcome they want]
Line 4: [Action -- what to do + why now]
```

## Story -- Hook, Backstory, Turning Point, Result, CTA

Best for: testimonials, case studies, UGC, transformation content

### Video Script Template (45-60s)
```
[0-3s]  HOOK: The result or transformation -- lead with the end state
[3-12s] BACKSTORY: Where they were before -- the struggle
[12-25s] TURNING POINT: What changed -- the moment they found the solution
[25-40s] RESULT: Specific, measurable outcomes -- numbers, timeline, proof
[40-45s] CTA: "If you want the same result..."
```

### Copy Template
```
Line 1: "[Result statement] -- and it only took [timeframe]."
Line 2: "Before, I was [struggling with X]..."
Line 3: "Then I [discovered/tried/started] [solution]..."
Line 4: "[Specific result with numbers]"
Line 5: "[CTA]"
```

## Before/After -- Contrast Framework

Best for: visual transformations, service showcases, results-driven offers

### Video Script Template (30s)
```
[0-3s]  HOOK: "Here is what [thing] looked like before vs after"
[3-10s] BEFORE: Show the problem state -- messy, broken, struggling
[10-20s] AFTER: Show the result -- clean, working, thriving
[20-25s] HOW: Brief explanation of what made the difference
[25-30s] CTA: "Want the same? [action]"
```

### Copy Template
```
Line 1: "Before: [problem state]"
Line 2: "After: [result state]"
Line 3: "[What we did to get there]"
Line 4: "[CTA]"
```

## Controversy -- Hot Take Framework

Best for: engagement, brand positioning, thought leadership, organic-feeling ads

### Video Script Template (30-45s)
```
[0-3s]  HOOK: Bold, contrarian statement that challenges a common belief
[3-15s] SUPPORT: 2-3 reasons why you believe this -- with evidence
[15-25s] REFRAME: What people should do instead
[25-30s] CTA: Soft CTA -- "follow for more" or "link in bio"
```

### Copy Template
```
Line 1: "[Contrarian statement]."
Line 2: "Here is why: [reason with proof]"
Line 3: "Instead, [what to do differently]"
Line 4: "[Soft CTA]"
```

## I Tested X -- Experiment Framework

Best for: demos, comparisons, tool reviews, educational content that converts

### Video Script Template (45-60s)
```
[0-3s]  HOOK: "I tested [X] for [timeframe] -- here is what happened"
[3-15s] SETUP: What you tested, why, and the rules
[15-35s] RESULTS: What actually happened -- show the data or demo
[35-45s] VERDICT: What you learned -- would you recommend it?
[45-50s] CTA: "Want me to set this up for you?" or "Try it yourself -- link below"
```

### Copy Template
```
Line 1: "I tested [X] for [timeframe]. Results:"
Line 2: "[Key finding 1]"
Line 3: "[Key finding 2]"
Line 4: "[Verdict + CTA]"
```

## Framework Selection Guide

| Audience Temp | Best Framework | Why |
|--------------|----------------|-----|
| Cold -- unaware | PAS | Starts with their pain, no trust needed |
| Cold -- problem-aware | Before/After | Visual proof is more convincing than words |
| Warm -- considering | AIDA | They are interested, build desire |
| Warm -- comparing | I Tested X | Proof and data help them decide |
| Hot -- ready to buy | Story | Social proof from someone like them |
| Any -- engagement play | Controversy | Gets shares and comments, builds audience |

## Timing Rules

- Hook MUST land in first 3 seconds -- if they scroll past, nothing else matters
- One idea per ad -- no tangents, no "also" or "by the way"
- 30 seconds is the sweet spot for cold traffic
- 45-60 seconds for warm traffic or story-based ads
- If the script reads longer than 60 seconds spoken, cut it down
